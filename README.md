README
===========================

###### 网站主旨
#### 此网站生动的描述了我国近年来在经济，生态，科技等领域取得的惊人成就，并且精选了许多我国近年以来重大的创新，突破事例。

###### 网站描述
#### 1.网站名称：
# **中国梦，复兴梦**

#### 2.网站布局：  
######## 打开page/index/index.html进入网站首页，
######## 网站的结构主要布局分为左侧抽屉式导航栏、右上角锚点导航和内容区；
######## 其中，左侧抽屉式导航栏是各个页面的跳转链接，方便浏览各个页面；
######## 其次右上角的锚点导航栏是为了方便提示浏览者阅读到哪一个版块并且浏览者想阅读某一个板块方便跳转；（为了美观并考虑到作用性，当尺寸为移动端设备时，该导航将会被隐藏）；
######## 网页内容区是展示网站内容的区域。

#### 3.网站概况：
######## 为了展示我国近年来的飞速发展和取得的惊人成就，提升大家爱国情怀和民族成就感、自豪感，增强对国家的希望，对党的信心，
######## 该网站分别分为了七个大类，从建设、科技、一带一路、长远的发展目光、思想政治、传统文化根基七个方面去体现我国从内而外的壮大，
######## 让人们对国家更加充满希望，充满信心，提升民族自豪感。

#### 4.操作介绍：
###### 每个页面的左上角的有个按钮，点击出现抽屉式导航，导航的链接可以跳转向其他页面，再次点击按钮可以关闭抽屉式导航，
###### 每个页面的右上角有提示导航，可以提示你当前位于该页面的哪一个板块，你也可以点击他跳转到当前页面你想看的板块，
###### 页面内容中有了解更多的按钮可以进行点击，帮助你了解更多该项目的内容，
###### 页面有可进行拖动的轮播图，你可以进行拖动，每一项都有精彩的内容供您阅览。

#### 5.技术：
###### 本网站主要使用了bootstrap、jQuery框架和fulPage.js、animate.css、WOW.js、scrollReveal.js、owl-carousel.js等插件，并使用了大量的html5和css3的新技术页面效果极为丰富
###### 布局上主要是用bootstrap进行布局和移动端设备的响应，网站内容主要用了fullPage.js全屏插件，使得网站能够更好的全屏进行展示，更加高端大气上档次；
###### 其次为了方便添加动画，使用了animate.css和wow.js进行搭配，使得页面的元素效果动感十足；
###### 为了更好的展示，光使用上面的动画搭配还不够，又使用了scrollReveal.js和css3的动画进行搭配，使得在回滚时也能出现一些必要的动画效果；
###### 为了增加一些鼠标特效，使用了html5的canvas，video，adiuo技术，使得一些必要的页面能有绚丽的效果，视频和音效营造网站的氛围；
###### 为了提升浏览的质量和效率，该网站不光有y轴浏览，在一些必要的板块加入了owl-carousel.js轮播插件，让网站的布局更加的精简，人性化，交互效果更好，使得网站多了一个X轴的浏览空间
###### 让浏览者能够横向看一些内容，而不是顺着Y轴从头看到尾，这样会比较的枯燥乏味，导致视觉疲劳；
###### 为了提升交互效果，该网站充满了许多的交互逻辑，有大量的css3的动画提示，二级弹窗等等交互，让浏览者能够对整个网站都充满着惊喜和探索的欲望，不容易觉得枯燥乏味，并且每一个交互
###### 逻辑都是很人性化的，每一个有交互的地方都有它交互的必要。

#### 6.设计：
###### 该网站从色彩，布局，内容，排版，交互都精心地进行了设计。
###### 色彩搭配使用了中国蓝为主色，体现的是中国近年来各个方面的快速发展，特写是科技这一方面，科技作为新时代的第一生产力是体验一个国家是否强大的重要体现，大到军事，小到生活，无处
###### 不充斥着科技这一元素。
###### 布局采用了全屏布局，抽屉式的导航可以减少页面空间的占用，作为一个展示性的网站能够更好的展示其内容，并且也能很好的完成页面之间跳转的交互。
###### 内容都是经过精心的挑选，从制造，科技，世界地位、可持续发展战略、中国梦的含义、中国传统文化的根基七个类别进行展示，体现的是中国不论是内在还是外在都是发展中强国，体现的是我国
###### 不仅仅注重外在工业、制造页，科技的发展，内在思想更是处于领先的水平。
###### 排版，采用了大量的留白，大图，大量的留白，内容却是丰富至极，要做到这点，就得精心的进行优化，每一段文字内容都是通过我们精心进行筛选优化的，这样才能让浏览者看到重点，而不是长篇
###### 大段的水文，简即是繁，简单的几个页面却花了我们无数的心思，网页在精不在多。
###### 交互，交互逻辑人性化，清晰，必要，没有多余的东西，该有动画交互的元素都是起到一定作用的，要么是营造网站氛围，或是让浏览者重点查看，引起浏览者的注意。
